# Simple Interest Calculator

A simple shell script project designed to calculate simple interest given principal, annual rate of interest, and time period in years.

## Description
This repository is created as part of a final project for getting started with Git and GitHub fundamentals. It contains a script that accepts parameters to compute simple interest formulas easily via the command line.
