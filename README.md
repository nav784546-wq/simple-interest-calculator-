# Simple Interest Calculator

## Description
This project contains a Bash script that calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

The calculator prompts the user to enter the principal amount, annual rate of interest, and time period, then displays the calculated simple interest.

## Features
- Calculates simple interest.
- Accepts user input for principal, rate, and time.
- Displays the result instantly.
- Simple and easy-to-use Bash script.

## Requirements
- Linux, macOS, or Windows with Git Bash
- Bash shell

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/github-final-project.git
   ```

2. Navigate to the project directory:
   ```bash
   cd github-final-project
   ```

3. Make the script executable:
   ```bash
   chmod +x simple-interest.sh
   ```

4. Run the script:
   ```bash
   ./simple-interest.sh
   ```

5. Enter:
   - Principal amount
   - Rate of interest
   - Time period

6. The script will display the calculated simple interest.

## Formula

```
Simple Interest = (Principal × Rate × Time) / 100
```
